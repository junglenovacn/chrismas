# Christmas Tree - Sci-Fi 3D Animated

一个炫酷的科幻风格3D圣诞树动画，支持自定义祝福文字分享。

## ✨ 特性

- 🎄 全息线框效果的3D圣诞树
- ⭐ 炫酷的星星粒子系统
- 🌈 赛博朋克风格配色
- 💫 动态光效和能量环
- 📝 支持自定义祝福文字
- 🔗 生成专属分享链接

## 🚀 快速开始

### 在线预览

直接在浏览器中打开 `index.html` 文件即可查看效果。

### 本地部署

1. 克隆或下载此项目
2. 用浏览器打开 `index.html`

## 📱 使用方法

### 生成祝福链接

1. 在页面底部输入框输入祝福文字（最多20字）
2. 点击"生成专属链接"
3. 复制链接分享给朋友
4. 朋友打开链接观看20秒后将看到你的祝福

### 分享链接格式

```
https://yourname.github.io/chrismas/?msg=圣诞快乐
```

## 🛠️ 技术栈

- Three.js - 3D渲染
- HTML5 Canvas - 星星纹理生成
- 原生 JavaScript - 无需框架
