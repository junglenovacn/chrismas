## 🎄 动画1：科幻圣诞树

一个炫酷的科幻风格3D圣诞树动画，支持自定义祝福文字分享。

### ✨ 特性

- 🎄 全息线框效果的3D圣诞树
- ⭐ 炫酷的星星粒子系统
- 🌈 赛博朋克风格配色
- 💫 动态光效和能量环
- 📝 支持自定义祝福文字
- 🔗 生成专属分享链接

### 🚀 访问方式

**在线预览**：https://junglenovacn.github.io/chrismas/christmas-tree.html

**本地预览**：用浏览器打开 `christmas-tree.html`

### 📱 使用方法

1. 在页面底部输入框输入祝福文字（最多20字）
2. 点击"生成专属链接"
3. 复制链接分享给朋友
4. 朋友打开链接观看20秒后将看到你的祝福

### 分享链接格式

```
https://junglenovacn.github.io/chrismas/christmas-tree.html?msg=圣诞快乐
```

## 🎊 动画2：中国春节 - 四合院

传统中国春节主题的3D四合院场景，包含完整的庭院结构和节日装饰。

### ✨ 特性

- 🏛️ **传统四合院结构** - 完整的四面围墙和传统大门
- 🏮 **大门口大红灯笼** - 两盏悬挂的巨型红灯笼
- 🎐 **院内灯箱阵列** - 两排传统纸质灯箱
- 💡 **院墙小彩灯** - 四面墙装饰的彩色小灯
- 🎆 **绚丽烟花系统** - 多彩爆炸效果
- 🔴 **传统中国配色** - 红色、金色与古典建筑风格
- 💫 **动态光效** - 所有灯光都有呼吸动画

### 🏠 场景布局

- **南墙（大门）**：传统大门结构，两侧各悬挂大红灯笼
- **北墙**：后墙装饰彩色小灯
- **东西墙**：两侧墙各悬挂6盏小灯笼 + 彩色小灯
- **院子**：8盏传统灯箱分两排排列
- **地面**：青石板纹理的庭院地面

### 🚀 访问方式

**在线预览**：https://junglenovacn.github.io/chrismas/spring-festival.html

**本地预览**：用浏览器打开 `spring-festival.html`

### 🎆 使用方法

- 拖动鼠标旋转视角（可以360°观察四合院）
- 滚轮缩放
- 点击"🎆 放烟花 🎆"按钮发射烟花
- 系统每2.5秒自动随机发射烟花

## 🏛️ 动画3：低多边形四合院

采用低多边形艺术风格的传统中式四合院，展现经典建筑的简约之美。

### ✨ 特性

- 🎨 **低多边形风格** - 简洁的几何体，卡通渲染感
- 🏠 **完整建筑布局** - 正房、厢房、倒座房、大门
- 🌅 **昼夜切换** - 点击按钮在白天和夜晚间切换
- 💡 **动态灯光** - 夜晚灯笼亮起，营造温馨氛围
- 🌳 **庭院元素** - 中央花坛、角落树木
- 🚶️ **交互功能** - 门可点击开合、自由视角控制

### 🏛️ 场景布局

#### 建筑结构
- **正房（北房）**：最高大的主屋，位于北侧，三间结构
- **东西厢房**：对称分布，各两到三间房
- **倒座房（南房）**：与正房相对，作为客房
- **大门**：东南角突出的门楼，红色大门结构

#### 连接与装饰
- **游廊**：连接所有房屋的有顶走廊
- **影壁**：大门入口内侧的装饰墙
- **花坛**：中央方形石台，种植彩色花朵
- **树木**：角落低多边形风格的装饰树木

### 🎮 交互功能

- **鼠标拖拽**：360°旋转观察四合院
- **滚轮缩放**：调整观察距离
- **右键平移**：移动观察视角
- **点击大门**：触发明门开合动画
- **昼夜按钮**：切换白天/夜晚模式

### 🌅 昼夜模式

#### 白天模式
- 明亮的蓝天背景
- 温暖的阳光从西南方照射
- 清晰的建筑阴影
- 灯笼熄灭

#### 夜晚模式
- 深蓝色夜空背景
- 蓝色月光照明
- 暗淡的环境光
- 红灯笼亮起，温暖照明

### 🎨 艺术风格

- **低多边形**：模型简洁，面数少，性能优化
- **纯色材质**：避免写实纹理，使用区块化色彩
- **卡通感**：轻微的抽象渲染风格
- **温暖色调**：午后暖光，宁静氛围

**在线预览**：https://junglenovacn.github.io/chrismas/siheyuan.html

**本地预览**：用浏览器打开 `siheyuan.html`
