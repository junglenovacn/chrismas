# 3D 节日动画

本仓库包含两个精美的3D节日动画页面：科幻圣诞树和中国春节主题。

---

## 🏠 首页导航

**在线预览**：https://junglenovacn.github.io/chrismas/

访问首页选择你想体验的节日动画！

---

## 🎄 动画1：科幻圣诞树

一个炫酷的科幻风格3D圣诞树动画，支持自定义祝福文字分享。

### ✨ 特性

- 🎄 全息线框效果的3D圣诞树
- ⭐ 炫酷的星星粒子系统
- 🌈 赛博朋克风格配色
- 💫 动态光效和能量环
- 📝 支持自定义祝福文字
- 🔗 生成专属分享链接

### 🚀 访问方式

**在线预览**：https://junglenovacn.github.io/chrismas/christmas-tree.html

**本地预览**：用浏览器打开 `christmas-tree.html`

### 📱 使用方法

1. 在页面底部输入框输入祝福文字（最多20字）
2. 点击"生成专属链接"
3. 复制链接分享给朋友
4. 朋友打开链接观看20秒后将看到你的祝福

### 分享链接格式

```
https://junglenovacn.github.io/chrismas/christmas-tree.html?msg=圣诞快乐
```

## 🎊 动画2：中国春节

传统中国春节主题的3D动画场景，包含红灯笼和烟花效果。

### ✨ 特性

- 🏮 悬浮的红灯笼阵列
- 🎆 绚丽的烟花效果
- 🔴 中国红与金色配色
- 💫 动态光效和粒子系统
- 🎇 自动和手动烟花发射

### 🚀 访问方式

**在线预览**：https://junglenovacn.github.io/chrismas/spring-festival.html

**本地预览**：用浏览器打开 `spring-festival.html`

### 🎆 使用方法

- 拖动鼠标旋转视角
- 滚轮缩放
- 点击"🎆 放烟花 🎆"按钮发射烟花
- 系统会自动随机发射烟花
